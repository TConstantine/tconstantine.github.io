Sunshine

Sunshine is a weather application that displays forecasts for the next 14 days. Includes High / Low temperatures, humidity levels, wind speed and atmospheric pressure. Ability to share the weather forecast with your friends and family via your own messaging / email apps. Ability to see the location you're getting the weather for on Google Maps. Daily notifications about the weather forecast for that day.

Forecasts are fetched from Open Weather Map

Github repository: Sunshine

  • Client: Personal project
  • Date: March 2017
  • Service: Android Development

Cineville

Cineville is an application that displays the most popular movies at any given time and the top rated ones. Includes information about the description of each movie, their release date, user rating, trailers and reviews. Users are also able to construct their own favorite list of movies.

Movie information is fetched from The Movie Database

  • Client: Personal project
  • Date: April 2017
  • Service: Android Development

Eleon

Eleon is a prototype of an olive oil wikipedia. Currently the application is an on-going project. Users are able to scan olive oil product codes and view information about the scanned olive oil, the corresponding manufacturer and its location. Alternatively users can also manually insert the codes.

  • Client: Panos Poursanidis
  • Date: May 2017
  • Service: Android Development
